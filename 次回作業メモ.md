### 2. Vercel公開APIでAI判定を再確認する

Vercelに `GROQ_API_KEY` を反映してRedeployした後、公開APIで確認する。

PowerShell例:

```powershell
$token = (Get-Content .env.vercel-import |
  Where-Object { $_ -match '^INTERNAL_API_TOKEN=' } |
  Select-Object -First 1) -replace '^INTERNAL_API_TOKEN=', ''

Invoke-RestMethod `
  -Method Post `
  -Uri "https://line-check-system.vercel.app/api/ai/route-messages" `
  -Headers @{ "x-internal-token" = $token } `
  -ContentType "application/json" `
  -Body '{"limit":3}'
```

期待:

- 500 ではなく JSON が返る。
- 未判定メッセージがあれば `routes` / `notifications` が増える。
- 既に判定済みしかなければ `processed` が少ない、または0でもよい。
