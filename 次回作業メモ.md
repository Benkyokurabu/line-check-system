### 1. 既存レコードの送信者名補完（次回最初にやること）

Vercelのdeployが完了したら以下を実行する（私が実行する）：

```powershell
$token = (Get-Content .env.vercel-import |
  Where-Object { $_ -match '^INTERNAL_API_TOKEN=' } |
  Select-Object -First 1) -replace '^INTERNAL_API_TOKEN=', ''

Invoke-RestMethod `
  -Method Post `
  -Uri "https://line-check-system.vercel.app/api/admin/backfill-display-names" `
  -Headers @{ "x-internal-token" = $token } `
  -ContentType "application/json" `
  -Body '{}'
```

期待レスポンス例：`{ "updated_users": 3 }` など

その後 `/dashboard` を開いて送信者名が表示されていることを確認する。
